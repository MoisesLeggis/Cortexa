## 💡 Visión del Proyecto

Crear una inteligencia artificial que no solo entienda **lo que escribes**, sino que también te ayude a comprender **lo que sientes** — fomentando la expresión emocional y el autoconocimiento.

---

## 🧠 ¿Qué hace actualmente?

| Función | Descripción |
|--------|-------------|
| 🔍 **Análisis emocional** | Detecta y clasifica emociones en cada entrada de texto |
| 📋 **Registro de estado** | Guarda el estado emocional del usuario en cada sesión |
| 📈 **Historial progresivo** | Construye una línea de tiempo emocional personalizada |

---

## 👤 Beneficio para el Usuario

Cortexa permite que cualquier persona pueda:

- 💬 Expresar libremente lo que siente
- 🔎 Entender sus emociones con mayor claridad
- 📊 Detectar patrones emocionales en su día a día
- 🗂️ Llevar un seguimiento personal sin esfuerzo

<div align="center">

> ✨ *Escribir deja de ser solo desahogo, y se convierte en **autoconocimiento**.*

</div>

## 👨‍⚕️ Enfoque Profesional — Valor Diferencial

Cortexa también está diseñado pensando en los **profesionales de la salud mental**.

Este sistema puede funcionar como una **herramienta complementaria para terapeutas**, proporcionando información objetiva basada en el registro del usuario.

### 📊 Próximas funcionalidades orientadas a terapia

- [ ] Análisis de emociones predominantes en periodos de tiempo
- [ ] Frecuencia de estados emocionales *(ej. ansiedad, tristeza)*
- [ ] Identificación de cambios emocionales relevantes
- [ ] Generación de informes emocionales del usuario
- [ ] Historial estructurado listo para revisión profesional

### 📈 ¿Por qué esto es importante?

En terapia, muchas veces el reto no es solo entender al paciente, sino tener **información constante y estructurada**. Cortexa permite:

- 🧩 Reducir la dependencia de la memoria del paciente
- 📐 Obtener datos más objetivos sobre su estado emocional
- 🔁 Detectar patrones que podrían pasar desapercibidos
- 🤝 Complementar sesiones con información real y continua

---

## 💬 Futuro: Asistente Emocional Inteligente

Cortexa evolucionará hacia un sistema conversacional capaz de:

- 🗣️ Mantener conversaciones naturales
- ❤️ Brindar apoyo emocional en tiempo real
- 🧘 Sugerir técnicas básicas *(respiración, reflexión, journaling)*
- 🎯 Adaptarse al estado emocional del usuario

<div align="center">

## 🚧 Estado del Proyecto

### ✅ Fase 1 — COMPLETADA

Se construyó la base completa del sistema:

**🧠 Inteligencia Artificial**
- Modelo de reconocimiento de emociones (NLP inicial)
- Integración con modelos de 🤗 Hugging Face
- Uso del modelo **BETO** (lenguaje en español 🇪🇸)

**📊 Datos**
- Recolección, limpieza y preparación de datasets
- Web scraping (Twitter, Reddit)
- Procesamiento con hilos *(extracción + traducción simultánea)*
- Entrenamiento en Google Colab

**⚙️ Backend**
- Backend con Django + REST API
- Base de datos con Neon ⚡
- Sistema de autenticación completo:
  - Registro de usuarios
  - Inicio de sesión
  - Verificación por correo
  - Recuperación y cambio de contraseña

**🎨 Frontend**
- Interfaz con HTML, CSS y JavaScript
- Demo funcional del análisis emocional

---

### 🔄 Fase 2 — EN DESARROLLO

Aquí es donde Cortexa evoluciona a algo **mucho más potente**:
